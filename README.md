# Enterprise Knowledge Assistant

An AI-powered Enterprise Knowledge Assistant built with FastAPI, LangChain, ChromaDB, and Large Language Models (LLMs).

## Features

- 📄 PDF Upload
- 📚 Automatic Document Processing
- ✂️ Smart Text Chunking
- 🔍 Semantic Search
- 🧠 Retrieval-Augmented Generation (RAG)
- 🤖 AI-powered Question Answering
- ⚡ FastAPI REST API
- 🐳 Docker Support (Coming Soon)

## Tech Stack

- Python
- FastAPI
- LangChain
- ChromaDB
- Sentence Transformers
- Hugging Face
- Docker
